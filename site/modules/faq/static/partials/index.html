Getting Started

The Joyent Cloud offers a high degree of security in various ways. Part of this security is tied into how you access your instances. Instead of a using a username/password combination for authentication, you will need to generate an SSH key, which is exponentially more secure. However, understanding how to work with SSH is a challenge if you are unfamiliar with the process.


This section will guide you through the process of signing up for the Joyent Cloud including walking you through generation of an SSH key.
