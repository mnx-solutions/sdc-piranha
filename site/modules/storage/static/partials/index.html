Introduction

Joyent Manta Storage Service is a cloud service that offers both storage and integrated compute for developers. Using Manta, developers can store and process any amount of data at any time, from any location, without requiring additional compute resources.

Getting started

There are two ways to get started

1. If you already have at least node.js v0.10 installed you can install the manta CLI and node SDK

npm install -g https://us-east.manta.joyent.com/manta/public/sdks/node-manta.tar.gz

2. If you do not have node.js installed you can use the installer for MAC

OSX Installer

CLI Example

Set environment
export MANTA_URL = https://us-east.manta.joyent.com
export MANTA_USER = {{account.login}}
export MANTA_KEY_ID = {{sshKeys[0].fingerprint }}
Create, list and delete objects
$ echo "Hello, Manta" > /tmp/hello.txt
$ mput -f /tmp/hello.txt /{{account.login}}/stor/foo
/{{account.login}}/stor/foo [==========================>] 100% 13B
$ mget /{{account.login}}/stor/foo
Hello, Manta
$ mls
foo
$ mrm /{{account.login}}/stor/foo
